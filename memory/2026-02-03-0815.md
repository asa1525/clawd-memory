## 📚 QVeris 使用方法

### **步骤 1：搜索工具**
```python
qveris_search(query="US stock NVDA price")
```

### **步骤 2：执行工具**
```python
qveris_execute(
  tool_id="binance.ticker.price.list.v3.xxx",
  search_id="...",
  params_to_tool='{"symbol": "NVDA"}'
)
```

在 `~/.openclaw/openclaw.json` 添加：
```json
{
  "tools": {
    "qveris": {
      "enabled": true,
      "apiKey": "qv_你的APIKey"
    }
  }
}
```
